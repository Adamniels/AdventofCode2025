# Advent of Code 2025

## Struktur
```
day01/
  solution.py      # Din lösning
  input.txt        # Din puzzle input
  example.txt      # Exempel från uppgiften
day02/
  solution.py
  input.txt
  example.txt
...
```

## Köra en dag
```bash
python day01/solution.py
```

## Debugga
1. Öppna `dayXX/solution.py`
2. Sätt breakpoints (klicka till vänster om radnumret)
3. Tryck **F5**
4. Debuggern startar och stannar vid dina breakpoints

## Skapa ny dag
Kopiera hela `day01/` mappen till `day02/`, `day03/` osv.
